## 🎯 Descripción

**Fangio Telecom** es un sistema integral de gestión de enlaces de telecomunicaciones que permite a los ingenieros y técnicos realizar análisis completos de factibilidad, gestionar bases de datos de enlaces, y generar reportes profesionales.

El sistema incluye:
- **Gestión de Enlaces PtP** (Point-to-Point)
- **Gestión de Enlaces PtMP** (Point-to-Multipoint)
- **Análisis de Perfil de Elevación**
- **Cálculos automáticos de factibilidad**
- **Dashboard interactivo con estadísticas**
- **Exportación de datos en múltiples formatos**

## 📦 Instalación

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a internet para APIs externas
- Cuenta de Firebase (para autenticación)

### Pasos de Instalación

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/fangio-telecom/enlaces-system.git
   cd enlaces-system
   ```

2. **Configurar Firebase**
   - Crear proyecto en [Firebase Console](https://console.firebase.google.com)
   - Habilitar Authentication con Email/Password
   - Copiar configuración a `config.js`

3. **Abrir en navegador**
   ```bash
   # Usando Python
   python -m http.server 8000
   
   # Usando Node.js
   npx serve .
   
   # O simplemente abrir index.html en el navegador
   ```

4. **Acceder al sistema**
   - URL: `http://localhost:8000`
   - Crear cuenta de usuario
   - Iniciar sesión

## 🎮 Uso

### 1. Inicio de Sesión
- Acceder a `login.html`
- Ingresar credenciales
- Sistema redirige automáticamente al dashboard

### 2. Gestión de Enlaces PtP
- **Formulario**: Ingresar datos de sitios A y B
- **Auto-cálculo**: Distancia y frecuencia se calculan automáticamente
- **Validación**: Sistema valida datos en tiempo real
- **Guardado**: Los enlaces se guardan en la base de datos local

### 3. Análisis de Factibilidad
- **Cálculos Automáticos**: Curvatura terrestre, zona de Fresnel
- **Validaciones**: Tipo de torre, estado operativo, transporte
- **Resultados**: Factibilidad con justificación detallada

### 4. Perfil de Elevación
- **Coordenadas**: Ingresar lat/lon de ambos sitios
- **Alturas**: Especificar altura de torres y RAN
- **Análisis**: Línea de vista, zona de Fresnel, curvatura terrestre
- **Visualización**: Gráfico interactivo con múltiples capas

### 5. Dashboard y Reportes
- **Estadísticas**: Métricas en tiempo real
- **Gráficos**: Distribución por factibilidad y frecuencia
- **Exportación**: Excel, CSV, PDF
- **Filtros**: Búsqueda y filtrado avanzado

### 📡 Módulo de Análisis de Microondas
```javascript
// Cálculo de path loss para microondas
const pathLoss = FANGIO_UTILS.calculateMicrowavePathLoss(distance, frequency, heightA, heightB, climate);

// Evaluación de calidad de señal
const quality = FANGIO_UTILS.evaluateSignalQuality(rssi, snr, ber);

// Selección automática de banda
const optimalBand = FANGIO_UTILS.selectOptimalFrequency(distance);
```

### 📊 Módulo de Monitoreo de Microondas
```javascript
// Monitoreo en tiempo real
```
